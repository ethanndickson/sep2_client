Notes: Current loading status of the file indicated by FileLink. This element MUST be set to one of the following values:<br/>0 - No load operation in progress<br/>1 - File load in progress (first request for file content has been issued by LD)<br/>2 - File load failed<br/>3 - File loaded successfully (full content of file has been received by the LD), signature verification in progress<br/>4 - File signature verification failed<br/>5 - File signature verified, waiting to activate file.<br/>6 - File activation failed<br/>7 - File activation in progress  <br/>8 - File activated successfully (this state may not be reached/persisted through an image activation)<br/>9-255 - Reserved for future use.<br/>
